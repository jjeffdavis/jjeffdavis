this page contains various experiments using the box model as well as floating text. (the tables have been copied from page 1).
Go ahead, stretch and shrink the width of this web page: many of the features are responsive.

.ooutline .cbox .clip-content

this box clips background to content. hence the padding, border and margin take the background color of the page.
the content takes the background color of the box, whose css features are given by class .cbox.
this example also has a white outline, which is completely outside of the box model. it is given by class .ooutline.
this is the only box on the page with an outline outside/beyond the box model.
the outline has its own border and margin.


.cbox .clip-padding .block-float-left

this text floats left. it only works when the page is fairly wide.
This should be replaced with responsive css flexboxes or css grid. For future research.

.cbox .text-center

class .text-center guarantees text centering. The picture and text below are in a table. Curiously, this works much better than the attempts to float text in the boxes above.
The table left centers, but curiously the text centers within column.
