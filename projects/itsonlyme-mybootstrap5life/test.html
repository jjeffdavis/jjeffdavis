About this page


This page is a fairly complicated mess of Bootstrap 5, HTML and CSS.

...The title, Not Waving But Drowning is a poem by Stevie Smith (England.) ...I started by creating a modest page squarely placed on Bootstrap so that this could be displayed on Androids as well as on large-screen Windows and Mac computers.

...The rest was fun, trying out all sorts of HTML and CSS to get everything centered in the column containers.

...It was no small chore to get the CSS to format attractive tables.

...The one part that doesn't work is the MiraCosta map. The width defaults to 300 px. ...At 300 px, it runs outside of its container for some widths. If I chose a smaller width, it would render the map uselessly small.
